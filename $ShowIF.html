$showIF

The `$showIF` function lets you hide or show an element based on the truthiness of the resting parameter provided.

 $showIF(restingParamter, onTruthyElement, onFalslyElement)

Here is an example:

Example - $showIF


app.LoadPlugin('Metro')
cfg.Portrait

class Main extends App {
  onStart() {
    this.layMain = ui.addLayout("main", "linear", "fillxy,vcenter");
    this.layMain.setChildMargins(0.02, 0.02, 0.02, 0.02);
    this.txt = ui.addText(this.layMain, "My Hybrid App");

    this.btn = ui.addButton(this.layMain, "First Btn", "primary");
    this.btn2 = ui.addButton(this.layMain, "Second Btn", "outline")
    $showIF(true, this.btn, this.btn2)   }

}
