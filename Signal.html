$signal

The `$signal` function lets you set signals to your variables and subscribe to value changes and set values.

The `$signal` function returns a setter/getter and a subscribe function.

 let theme = $signal('light')

Here is an example featuring a theme based signal:

Example - $signal


app.LoadPlugin('Metro')
cfg.Portrait

class Main extends App {
  constructor() {
   super()
   this.theme = $signal('light')
   this.theme.subscribe(this.onTheme)
  }
  onStart() {
    this.layMain = ui.addLayout("main", "linear", "fillxy,vcenter");
    this.layMain.setChildMargins(0.02, 0.02, 0.02, 0.02);
    this.txt = ui.addText(this.layMain, "My Hybrid App");

    this.btn = ui.addButton(this.layMain, "Switch Theme", "primary");
    this.btn.on('click', this.onClick)
  }

  onTheme(theme) {
    ui.setTheme(theme)   }
  onClick(e) {
    e.stopPropagation()
    if (this.theme.value === 'light') {
      this.theme.value = 'dark'
    } else this.theme.value = 'light';
  }
}
