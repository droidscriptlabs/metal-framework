$component

The `$component` function lets you author your own components without having to write verbose class code.

After that you will be exposed to the batch and css methods.

 $component(parent, tag, width, height, properties, options)

The properties param is an object that contains the properties of the html element and use direct dom access. So you can use DOM Api's like textContent.

Here is an example:

Example - $component


app.LoadPlugin('Metro')
cfg.Portrait

class Main extends App {
  constructor() {
    super();
    this.layMain = ui.addLayout("main", "linear", "fillxy,vcenter");
    this.txt = ui.addText(this.layMain, "My Hybrid app");
  }

  onStart() {
    this.layMain.setChildMargins(0.02, 0.02, 0.02, 0.02);

    let btn = ui.addButton(this.layMain, "My Button", "primary");

    let outlinebtn = $component(this.layMain, "button", -1, -1, {
      textContent: "Hello World",
    }).css`
    border: 2px solid #6200ea;
    color: #6200ea;
    background-color: transparent;
    font-family: "Archivo", sans-serif;
    font-weight: 500;
    font-size: 1rem;
    text-align: center;
    cursor: pointer;
    padding: 0.5rem 1rem;
    transition: background-color 0.3s, color 0.3s;
      
      &:hover {
        background-color: #6200ea;
        color: white;
    }

    &:active {
      background-color: #3700b3;
      border-color: #3700b3;
    }`.on("click", function (e) {
      e.stopPropagation();
      ui.showPopup("Clicked Custom Button");
    });

  }
}
